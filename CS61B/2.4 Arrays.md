1. System.arraycopy(b, 0, x, 3, 2)
	各参数意义：**b**：复制来源	**0**：开始复制的位置	**x**：目标	**3**：开始粘贴的位置		**2**：复制的个数

2. 当数组到达存储上限时，重新建立一个更大的数组，将原来数组的内容复制过去并添加新的内容
```
int[] a = new int[size + 1];
System.arraycopy(items, 0, a, 0, size);
a[size] = 11;
items = a;
size = size + 1;
```
