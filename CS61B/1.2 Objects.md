1. 静态方法无法被instance调用
2. class数组`Dog[] dogs = new Dog[2];   dogs[0] = new Dog(2);`

3. class method如`x = Math.sqrt(100);`，Instance methods如`dan.makenoise();`

4. this指代当前调用该方法的object: 
```
public Dog maxDog(Dog d2) {
    if (this.weightInPounds > d2.weightInPounds) {
        return this;
    }
    return d2;
}
```
