1. extends继承父类的时候会继承
-   All instance and static variables
-   All methods
-   All nested classes
- 不会继承constructor
- private methods不可被子类调用

2. super调用父类的方法：
```java
@Override  
public Item removeLast() {  
    Item x = super.removeLast();  
    deletedItems.addLast(x);  
    return x;  
}
```
