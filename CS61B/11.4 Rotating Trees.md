**左旋（Left Rotation）**：

-   将节点的右子节点提升为新的根节点。
    
-   原根节点成为新根节点的左子节点。
    
-   新根节点的左子节点（如果有）成为原根节点的右子节点。
```java
Node leftRotate(Node x) {
    Node y = x.right;
    x.right = y.left;
    y.left = x;
    return y;
}
```
**右旋（Right Rotation）**：

-   将节点的左子节点提升为新的根节点。
    
-   原根节点成为新根节点的右子节点。
    
-   新根节点的右子节点（如果有）成为原根节点的左子节点。
```java
Node rightRotate(Node y) {
    Node x = y.left;
    y.left = x.right;
    x.right = y;
    return x;
}
```
